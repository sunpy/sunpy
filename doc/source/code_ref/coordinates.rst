SunPy coordinates
=================

The SunPy coordinates submodule is an implementation of the common solar
physics coordinate frames in to the Astropy coordinates system. 
See :ref:`astropy:astropy-coordinates` for more information.

.. warning::

    This submodule is in beta, the accuracy of the transformations have not 
    been verified. See `sunpy.wcs` for the previous version of these 
    transformations.
