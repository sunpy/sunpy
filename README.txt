======================================================================

DESCRIPTION:

SunPy is a Python library for working with solar physics data.
We are currently just getting started with the effort so there
is not much to see just yet.

To request a new feature or to report a bug, visit 
  
    https://github.com/sunpy/sunpy

To find out more information about the project visit:
  
    http://sunpy.org
    
======================================================================

REQUIREMENTS:

    * Python (2.7+)
    * pyfits
    * numpy
    * matplotlib
    * scipy
